# Climate Change Analysis and Prediction using Machine Learning

## 📌 Project Overview
This project analyzes historical climate data and predicts future temperature trends using machine learning techniques.

## 📊 Features
- Time-series climate data analysis
- Feature engineering (lag, rolling mean)
- Machine learning prediction (Linear Regression)
- Future temperature forecasting
- Power BI dashboard for visualization

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Power BI

## 📁 Dataset
- ERA5 Climate Dataset (1950–2023)
- Variables: Temperature (tas), Precipitation (pr), Humidity (hurs)

## 🤖 Model
- Linear Regression
- Time-series feature engineering

## 📈 Results
The model successfully predicts temperature trends and shows good alignment with actual data.

## 🚀 Future Scope
- Use advanced ML models (XGBoost, LSTM)
- Add more climate variables
- Improve prediction accuracy
